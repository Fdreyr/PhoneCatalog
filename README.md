# PhoneCatalog

## How to run

**Make sure your NPM version is >=7.0.0**

Clone the repository

```bash
git clone https://github.com/Fdreyr/PhoneCatalog.git
```

Install the dependencies

```bash
npm install
```

Build the code
```bash
npm run build
```

Run the server
```bash
npm run start
```

Browse to `http://localhost:3250/` in your browser.
